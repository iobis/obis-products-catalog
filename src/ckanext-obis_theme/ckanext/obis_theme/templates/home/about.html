    Discover OBIS-mediated Information Products

    The OBIS Data Products Catalog helps you find and access information products built from OBIS-mediated marine biodiversity data. As our network has grown, we've found that it was challenging to keep track of all the amazing products that have been created from OBIS-mediated data. This metadata catalog helps users find and access those products.

    What's in the Catalog

    Our catalog includes diverse products created from OBIS-mediated data:

    • Maps
    • Models
    • Dashboards
    • Data visualizations
    • Infographics
    • Derived datasets (e.g. improved for a specific purpose)
    • Reports and Presentations

    Each product includes detailed metadata about data sources, methods, authors, and how to access or cite the work.

    Linked Data

    We try as much as possible to use linked data so that the people who created the products don't have to do extra work to maintain metadata. We pull institution records from the OBIS and OceanExpert APIs. We have a tool for import of Zenodo records via a DOI.

    Community Effort

    This catalog is developed through the OBIS Products Coordination Group (PCG). The PCG is tasked with coordinating data and information products, from any type of analysis (e.g. description, data visualisation, etc.) that synthesise and generate new information from data hosted on OBIS and other sources. The PCG meets every other month and is open to attendance by the OBIS community. More information on the PCG can be found in the [OBIS Manual](https://manual.obis.org/nodes.html#obis-products-coordination-group), and by contacting the OBIS Helpdesk (helpdesk@obis.org).
